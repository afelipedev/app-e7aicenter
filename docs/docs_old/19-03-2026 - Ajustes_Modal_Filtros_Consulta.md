## Implementação

Foram aplicados ajustes finos de layout e design no modal de filtros das consultas para melhorar espaçamento, contraste visual e hierarquia tipográfica.

## O que foi ajustado

- Refinamento do cabeçalho do modal com melhor contexto visual e hierarquia de entrada.
- Reorganização dos filtros em blocos visuais com separação mais clara entre tags, critérios principais e contexto complementar.
- Ajustes nos labels, textos auxiliares e ritmo vertical para facilitar leitura e escaneabilidade.
- Melhoria visual dos chips selecionados, botão de adicionar opções e campos de data.
- Ajuste dos botões finais de ação com acabamento mais consistente com o restante da feature de processos.

## Arquivo alterado

- `src/features/processes/components/ProcessFiltersSheet.tsx`

## Validação

- `ReadLints` sem erros no arquivo alterado.

## Observação

Como o componente é compartilhado entre as telas de consultas, o ajuste visual passa a valer para os fluxos que reutilizam esse modal.
