# _Endpoints_

A API Pública do Datajud oferece várias rotas para pesquisa de informações processuais devido à natureza e organização do Judiciário brasileiro.

A url principal de acesso é a url [https://api-publica.datajud.cnj.jus.br/](https://api-publica.datajud.cnj.jus.br/) e deverá ser seguida de um aliase correspondente ao Tribunal que deseja obter os dados processuais.

Exemplo

O endpoint do Tribunal Regional Federal da 1ª Região é **[https://api-publica.datajud.cnj.jus.br/api\_publica\_trf1/](https://api-publica.datajud.cnj.jus.br/api_publica_trf1/)**.
